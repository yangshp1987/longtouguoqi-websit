## 🆘 三大高频救援场景

### 场景 1：网站打不开

```
1. 先用手机 4G 流量打开 www.longtouguoqi.com.cn
   确认是否真打不开（避免是自己 WiFi 问题）

2. 如果真打不开，按顺序检查：
   
   ✓ 打开 GitHub 仓库页面
     https://github.com/<账号>/longtouguoqi-website
     看最近一次"绿色√"提交是什么时候
   
   ✓ 如果 30 分钟内有过提交 → 大概率是改坏了
     → 立刻执行下面"一键回滚"

3. 如果不是改动造成的 → 立刻联系技术顾问
```

### 场景 2：一键回滚（改坏了怎么救）

```
Step 1：打开网页
   https://github.com/<账号>/longtouguoqi-website/commits/main

Step 2：找到"改坏之前"的那条记录
   （看日期，比如今天是 4月19日，就找 4月18日及以前的）

Step 3：点击那条记录右边的 [Revert] 按钮
       
Step 4：GitHub 跳转新页面，依次点击：
       [Create pull request]
       → [Merge pull request]
       → [Confirm merge]

Step 5：等 1-2 分钟，刷新网站（Ctrl + F5）
        ✅ 网站应该恢复正常
```

> ⏱️ 整个过程 3 分钟内完成  
> 💡 如果不会操作，手机拍屏幕发微信给技术顾问

### 场景 3：域名续费 / 备案问题

```
⚠️ 这是最容易忘记、后果最严重的事！

每年检查 2 次：1 月 1 日、7 月 1 日

登录腾讯云控制台 → 域名管理 → 查看到期时间

1. 域名到期：提前 30 天在腾讯云续费（几十元一年）
2. SSL 证书到期：EdgeOne 自动续期，不用管
3. ICP 备案：实名信息变更时必须更新
   - 法人变更 → 30 天内重新提交备案
   - 公司注销 → 网站必须下线或转移
```

## 📝 日常维护操作速查

| 想做什么 | 打开 CodeBuddy IDE 对 AI 说 |
|---------|------------------------------|
| 改电话号码 | "把首页电话 xxx 改成 xxx，所有地方都要改" |
| 改地址 | "把联系页的地址改成 xxxx" |
| 加一条新闻 | "在新闻页顶部加一条：标题 xxx，日期 xxx，内容 xxx，来源 xxx" |
| 换首页大图 | 先拖新图到 images/ → "把首页 hero 图换成 xxx.jpg" |
| 改产品价格 | "把产品页 xxx 的价格从 xxx 改成 xxx" |
| 加一个店铺链接 | "在首页渠道区加一个 xxx 链接，地址是 xxx" |

改完别忘记 3 步：
1. 点"接受更改"
2. 左下角 Git：写提交消息（今天日期+改了啥）
3. 点 Commit ✓ 和 Push ⬆

## 💡 快速反应手册

```
情况：刚才改了点东西，网站白屏了！
  → 不要慌 → 打开 GitHub → 一键 Revert（见场景 2）

情况：AI 改出来的东西我不满意
  → CodeBuddy 里点"拒绝更改" → 重新用更清楚的语言告诉 AI

情况：图片上传了但网站没显示
  → 检查：文件名是否带中文或空格？改成纯英文+数字

情况：Commit 按钮点不了
  → 先在"消息框"里填写"做了什么改动"

情况：收到陌生邮件说要"续费域名 xxx 元"
  → 99% 是骗子！只登录腾讯云官网确认，不要点邮件里的链接
```
