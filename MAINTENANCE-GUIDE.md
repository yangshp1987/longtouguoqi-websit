## 一、方案选型分析

### 1.1 WorkBuddy vs CodeBuddy IDE —— 哪个更适合？

| 对比维度 | WorkBuddy | CodeBuddy IDE（VS Code插件） |
|:---|:---|:---|
| **定位** | AI办公助手（Excel/PPT/文档） | AI编程助手（代码项目） |
| **改HTML代码** | ⚠️ 能改，但不专业 | ✅ 专门为此设计 |
| **代码实时预览** | ❌ 没有 | ✅ 改完立刻看到网页效果 |
| **版本回滚** | ❌ 没有 | ✅ Git一键回滚 |
| **部署上线** | ❌ 不支持 | ✅ 可直接部署 |
| **文件同步** | ❌ 只能微信传文件 | ✅ Git自动同步 |
| **学习成本** | ⭐ 极低 | ⭐⭐ 低（需学3个按钮） |
| **适合维护网站** | ❌ 不适合 | ✅ 非常适合 |

**结论：维护网站必须用 CodeBuddy IDE，但我们把 Git 操作简化到极致。**

> 💡 WorkBuddy 可以作为辅助工具——比如让它帮忙写新闻文案、整理产品描述文字，然后粘贴到 CodeBuddy 里改代码。

## 四、小白操作手册（打印给他贴墙上）

### 🎯 日常操作只需4步，每步都有截图位置标注

---

### 第①步：打开项目

1. 双击桌面上的 **VS Code** 图标
2. 文件 → 打开文件夹 → 选择 `longtouguoqi-website` 文件夹
3. 左侧就能看到所有网页文件

---

### 第②步：告诉AI你要改什么

1. 点击 VS Code **左侧边栏的 CodeBuddy 图标**（一个AI小logo）
2. 在对话框里**用大白话说你要改什么**，比如：

**📞 改联系方式：**
```
帮我把网站里所有的联系电话从 13072913676 改成 13999999999，所有文件都要改。
```

**📰 添加新闻：**
```
在 news.html 里添加一条新闻：
- 标题：龙头国漆参加2026西安非遗博览会
- 日期：2026年5月15日
- 内容：龙头国漆携精品漆器亮相西安非遗博览会，展示了……
风格和现有新闻保持一致。
```

**🛍️ 改产品信息：**
```
在 products.html 里把精制推光漆的价格从168元改成188元。
```

**🔗 加社交账号：**
```
在所有页面的底部添加小红书账号，名字叫"龙头国漆"，链接是 https://www.xiaohongshu.com/xxx
```

3. AI 会自动修改代码，你能看到**改了哪里**（绿色=新增，红色=删除）
4. 点击 **"接受"** 应用修改，或 **"拒绝"** 撤销

---

### 第③步：预览看看改对没有

1. 在左侧文件列表里**右键点击 index.html**
2. 选择 **"Open with Live Server"**
3. 浏览器会自动打开网站预览
4. 看看改的地方对不对，其他页面也点进去检查一下

> 💡 如果不对，直接告诉 CodeBuddy "刚才改的不对，帮我撤销" 或者按 `Ctrl+Z` 撤销

---

### 第④步：上传改动（最关键的3个按钮）

```
📌 记住口诀：左三 → 写字 → 勾勾 → 推推
```

1. 点击 VS Code **左侧第三个图标**（看起来像树枝分叉的 Git 图标）
2. 你会看到"更改"列表，这些就是你刚才改的文件
3. 在上方**消息框**里写一句话说明改了什么，比如：`更新了联系电话`
4. 点击 **✓ 勾号按钮**（提交）
5. 点击 **"同步更改"** 或 **↑ 推送按钮**

✅ 完成！改动已上传到GitHub，技术负责人那边就能看到了。

---

### ⚠️ 每次开始改之前，先点一下"同步"

> 打开 VS Code 后，先点击底部状态栏的 **🔄 同步图标**，确保你的文件是最新的。
> 这样可以避免和别人的改动冲突。



等小白操作熟练后，可以设置自动化：
- 他一推送到 GitHub main 分支

### 方案C：加上审批流程（最安全）

- 小白的改动自动创建 Pull Request
- 你在手机GitHub App上审批（一键操作）
- 审批通过后自动部署

## 十、常见问题（Q&A）

**Q: 小白改坏了网站怎么办？**
A: 他只是改了代码推到GitHub，网站不会自动更新。你review没问题后再部署。即使部署了有问题，`git revert` 一键回滚。

**Q: 两个人同时改了怎么办？**
A: Git会自动合并。如果改了同一行，会提示"冲突"，到时候你来解决就行（这种情况很少）。

**Q: CodeBuddy是免费的吗？**
A: 是的，基础功能完全免费，够用了。

**Q: 如果运维人员换人了怎么办？**
A: GitHub上移除旧协作者、添加新协作者，新人装好软件克隆仓库就行。代码和历史记录都在GitHub上，不依赖任何个人电脑。

**Q: 以后想加图片/视频怎么办？**
A: 告诉CodeBuddy "帮我在xx页面添加一张图片"，把图片文件放到项目文件夹里，AI会自动写好代码引用它。
